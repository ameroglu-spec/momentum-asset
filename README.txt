Momentum Asset V5

Kurulum:
1) Supabase SQL Editor içinde supabase_setup_v5.sql dosyasını çalıştırın.
2) momentum_asset_v5 klasörünü Netlify Drop'a yükleyin.
3) Uygulamaya giriş yapıp mevcut verilerinizi kontrol edin.

V5 yenilikleri:
- Profesyonel dashboard
- Finans takvimi ve 30 günlük özet
- Belge Merkezi
- Global arama
- Hızlı işlem menüsü / mobil FAB butonu
- Light/Dark tema
- PWA manifest ve temel offline cache
- V4.1 güvenli dosya adı düzeltmesi ve V4.2 belge-kayıt bağlantısı korunmuştur.

Not: V4.2 kullanıyorsanız tablo yapısı büyük ölçüde aynıdır; yine de supabase_setup_v5.sql dosyasını çalıştırmak güvenlidir.
