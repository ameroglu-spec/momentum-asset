# Momentum Hub

**Know Today. Plan Tomorrow.**

Momentum Hub; ev, araç, gelir/gider, belge, takvim, bildirim ve raporları tek merkezden takip etmek için geliştirilen kişisel kontrol merkezidir.

## Current Version

**V6.0 Stable**

## Core Modules

- Bugün
- Momentum AI
- Varlıklar
- Araçlar
- Belgeler
- Raporlar
- Takvim
- Bildirimler
- Tanımlar
- Yedek

## Deployment

Proje GitHub üzerinden Netlify'a bağlıdır. Değişiklikler GitHub Desktop ile commit ve push edildiğinde Netlify otomatik deploy eder.
